# 知会

  1. 本网站同时提供了英文版 (opens new window), 如果中英文版本有出现不一致的地方，以英文版为准。
  2. aiM18平台提供了两种开发模式,分别是 App开发 和 API集成.
  3. 关于 App开发, 你需要了解aiM18 App的定义和它的组成结构. 我们推荐你按顺序阅读App开发相关的内容.
  4. 关于 API集成, 你需要先了解 aiM18 API是如何授权访问的, 然后你可以通过顶部的搜索栏查找相应的API信息。
