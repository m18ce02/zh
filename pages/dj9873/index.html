API授权流程

# 知会

所有应用程序接口调用都要求您事先在aiM18平台注册并获得授权。

# 注册应用程序

在 aiM18平台的[应用授权列表]模块注册你的应用程序，得到Client ID 和 Client Secret。

# 获取访问令牌

通过使用Client ID、Client Secret以及可以正常登录到aiM18的用户账号和密码来获取访问令牌(access_token)。

(请注意：密码需要用SHA1进行加密)

欲了解更多详情，请查看 aiM18 访问授权
